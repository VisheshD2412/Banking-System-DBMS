# Banking System DBMS Project

## Project Overview
A comprehensive banking database management system implementing core DBMS concepts including ER modeling, complex SQL queries, normalization, transaction management, and concurrency control.

## Setup Instructions
1. Install MySQL/PostgreSQL
2. Run module1/schema.sql to create database
3. Run module1/sample_data.sql to populate tables
4. Execute queries from module2/queries.sql

## Modules Description
- **Module 1**: ER diagram design and relational schema conversion
- **Module 2**: Complex SQL queries with joins, subqueries, and aggregations
- **Module 3**: Functional dependency analysis and normalization
- **Module 4**: ACID properties implementation with transaction control
- **Module 5**: Lock-based concurrency control demonstrations
- **Module 6**: Hybrid SQL-NoSQL integration with analytics

## Technologies Used
- MySQL/PostgreSQL
- MongoDB (Module 6)
- Python/Flask (API implementation)
- Git/GitHub for version control
